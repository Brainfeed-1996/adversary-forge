# Architecture

## Overview

Adversary Forge evaluates AI applications and agentic systems under adversarial conditions.

## Main components

- scenario registry
- target adapters
- execution harness
- policy and safety evaluator
- scoring engine
- reporting and dashboard layer

## Execution flow

1. load scenario pack
2. configure target adapter
3. run attack sequence
4. collect responses, tool calls, and traces
5. evaluate outcomes
6. score and store results
7. generate comparative reports
